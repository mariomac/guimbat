# Guimbat

A terminal-style calculator powered by the [Numbat](https://numbat.dev) expression language, built with [iced](https://github.com/iced-rs/iced).

## Building

```sh
cargo build --release
```

## Running

```sh
cargo run --release
```

## Packaging as a macOS .app bundle

Install `cargo-bundle` once:

```sh
cargo install cargo-bundle
```

Then build and bundle:

```sh
cargo bundle --release
```

The `.app` will be created at:

```
target/release/bundle/osx/Guimbat.app
```

### Notes

- **Icon** — `assets/icon.icns` is included and already referenced in `Cargo.toml`. It was generated by `assets/make_icon.swift`, a Core Graphics script that draws original artwork (dark calculator with orange operator keys and a green display). The artwork is released under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) — no attribution required. To regenerate or customise it, run:
  ```sh
  cd assets && swift make_icon.swift
  mkdir -p AppIcon.iconset
  for size in 16 32 64 128 256 512; do
    sips -z $size $size icon_1024.png --out AppIcon.iconset/icon_${size}x${size}.png
    sips -z $((size*2)) $((size*2)) icon_1024.png --out AppIcon.iconset/icon_${size}x${size}@2x.png
  done
  iconutil -c icns AppIcon.iconset -o icon.icns
  ```
- **Bundle identifier** — the `identifier` field (`com.guimbat.app`) is the reverse-DNS bundle ID used by macOS. Change it to match your domain if distributing.
- **Code signing** — `cargo-bundle` does not sign the bundle. For distribution outside of direct download, run `codesign` manually after bundling.
